Looper

This is a tool for making music together in times of quarantine. Still work in progress. To try anyway (no guarantees!), proceed as follows:

  1. Use headphones.
  2. Open the session manager in a new tab.
  3. Click the start button and copy the session ID to the clipboard.
  4. Open the client in a new tab. (If you can memorize the rest of these instructions, same tab is fine.)
  5. Enter the session ID and click start. Now you can already rock solo.
  6. Share the session ID with your friends and play together.

To silence the beast, just close or reload the tabs.
